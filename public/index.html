Image Search Abstraction Layer

USAGE:

  1. Search for images passing a search string in the URL, using the /search/:searchString path. The API will return the image URLs, snippets and page URLs of the first 10 search results.
  2. To paginate through the results, append an offset parameter to the URL (e.g.: ?offset=11).
  3. To view a list of the 10 most recently submitted search string, use the /recentsearches path.
